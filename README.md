# 🧾 Retail Billing System 

## 📌 Overview
This is a **Retail Billing System** developed using Java in Apache NetBeans IDE.  
The project is designed to manage billing operations, product handling, and basic admin functionalities for a retail shop.

---

## 🚀 Features
- Admin Login System
-  Product Management & Category Management
-  Bill Generation
-  Billing History
-  File Handling for Data Storage
- Java Swing-based UI
---

## 🛠 Technologies Used
- Java (JDK 24)
- Swing (GUI)
- MySQL (Database)
- JDBC (Database Connectivity)
- Apache NetBeans IDE

---

## ⚙️ Setup
1. Clone repo  
```bash
git clone https://github.com/mahima68/Retail-Billing-System.git
```  
2. Open in NetBeans
3. Configure MySQL in DBLoader.java
4. Add MySQL connector JAR
5. Run project (F6)
   
---

## 📁 Structure

```bash
src/        → source code  
myuploads/  → product images  
libs/       → MySQL connector
```
